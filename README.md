# Name

Battery Finder Program

# URL

https://yasuhironagahama.github.io/battery-finder-program/

# Explain

電池のデータセット（電池名、容量（Ah）、電圧、最大放電電流（A）、終止電圧）、カメラのデータセット（メーカー、製品名、消費電力（Wh ワット時間））、選択されたブランド、モデル、パワーをもとに、使用できるバッテリーのリストを表示します。
